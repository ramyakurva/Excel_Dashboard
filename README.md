# Excel_Dashboard
📊 Interactive Super Store Sales Dashboard

This project is an Interactive Sales Dashboard created in Microsoft Excel using Super Store sales data. I built this dashboard as part of my Data Analytics learning journey through Skill Course by Satish Dhawale.

The dashboard helps users analyze sales performance and gain business insights through interactive visualizations and filters. Using Pivot Tables, Pivot Charts, Slicers, and KPI Cards, I transformed raw data into a clear and easy-to-understand dashboard.

Key Insights Included:
1. Total Sales Analysis
2. Average Sales Analysis
3. Total Transactions Count
4. Total Quantity Sold
5. Ship Mode Sales Distribution
6. Region-wise Transaction Count
7. Month-wise Quantity Sold Trend
8. Top 5 Cities by Sales
9. Category-wise Transaction Count
10. Consumer Type-wise Quantity Ordered

Dashboard Highlights:
